# Chrysalis Download Bundle

This folder contains non-malware tooling and report artifacts referenced in the Chrysalis blog post.

- `scripts/`: unpacking, emulation, rendering, and pipeline orchestration scripts
- `ida/`: IDAPython automation scripts
- `notebooks/`: analysis notebooks and supporting notebook data
- `input/`: hash manifest only (`input_sha256.csv`)
- `reports/binary_diff/`: patched PE diff outputs (`.txt` and `.json`)
- `reports/db_diff_reports/`: CSV outputs from sqlite diff analysis
- `docs/`: flowchart source and rendered image

Note:
- Original malware samples and malware-derived binaries are not redistributed in this repository.
